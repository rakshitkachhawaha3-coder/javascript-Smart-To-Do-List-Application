# ✅ Smart To-Do List v1.0

A modern and responsive To-Do List application built using **HTML, CSS, and JavaScript**. This project focuses on practicing core JavaScript concepts by creating a fully functional task management application.

---

## 🚀 Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Search tasks instantly
* Filter tasks (All, Completed & Pending)
* Automatically save tasks using Local Storage
* Dark Mode support
* Responsive design for desktop and mobile devices

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

---

## 📚 Concepts Practiced

* DOM Manipulation
* Event Listeners
* Functions
* Arrays & Objects
* JSON
* Local Storage
* CRUD Operations
* Dynamic UI Updates
* Responsive Web Design

The main objective of this project was to gain practical experience with JavaScript by building a real-world application instead of only studying theory. It demonstrates how JavaScript can be used to create interactive, user-friendly web applications.
